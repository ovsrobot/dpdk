ZTE Storage Data Accelerator (ZSDA) Poll Mode Driver
==================================================

The ZSDA crypto PMD provides poll mode Cipher and Hash driver
support for the following hardware accelerator devices:

* ``ZTE Processing accelerators 1cf2``
