R8169 Poll Mode Driver
======================

The R8169 PMD provides poll mode driver support for Realtek 2.5 and 5 Gigabit
Ethernet NICs.

Features
--------

Features of the R8169 PMD are:

* Checksum offload
* TCP segmentation offload
* Jumbo frames supported
