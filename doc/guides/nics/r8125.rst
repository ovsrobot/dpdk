R8125 Poll Mode Driver
====================

The R8125 PMD provides poll mode driver support for Realtek 2.5 Gigabit
Ethernet NICs.

Features
--------

Features of the R8125 PMD are:

* MAC/VLAN filtering
* Checksum offload
* TCP segmentation offload
* Jumbo frames supported
