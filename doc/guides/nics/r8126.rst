R8126 Poll Mode Driver
======================

The R8126 PMD provides poll mode driver support for Realtek 5 Gigabit
Ethernet NICs.

Features
--------

Features of the R8126 PMD are:

* MAC/VLAN filtering
* Checksum offload
* TCP segmentation offload
* Jumbo frames supported
